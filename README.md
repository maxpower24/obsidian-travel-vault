# Obsidian Travel Vault

The first part of my modular obsidian vault. This section focuses on a travel planner and itinerary.


## CSS Snippets

**full-width:** Use the full width of the page.
**travel-gallery:** Styles the bases card view for the travel vault.

## Core Plugins

Aside from the default core plugins, workspaces should be enabled.

## Community Plugins

1. Bases card redirect - https://github.com/maxpower24/bases-card-redirect
    Pending upload to the community plugin list. Redirect bases card view entries via frontmatter links.
2. Folder notes - Notes for folder level dashboards
3. Buttons - self explanatory
4. Map view - generating the google place notes as well as map functionality for trips
5. Templater - we will be utilizing some scripts and advanced templates via templater

## Folder Structure
Default folder structure can be tweaked slightly.

Travel folder will be defined in a config file so needs to be adjusted if deviating from the repo folder structure.

Google places folder will be defined in map view settings, so similarly should be updated if deviating from repo.

Meta folder might be a bit more complex, tweaking templater, templates and dashboards if deviating from repo.

## Usage
The vault contains some example data, feel free to delete any notes folders in the Trips folder as well as any notes in the Locales, Places and Countries folders.

Move any folders around as per the above folder structure section and ensure you update the config file and plugin settings as mentioned. Update community plugin settings like the google api key.

Now you can begin creating your first trip! Make sure to give it a unique name
